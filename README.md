```

### Run any script:

bash scripts/manifest_generator.sh
```

---

## 📜 Script Descriptions

| Script | Description |
|--------|-------------|
| system_info.sh | Displays username, hostname, uptime, kernel version, and OS name |
| package_checker.sh | Checks if Git is installed and shows version/location |
| disk_auditor.sh | Shows disk usage and file permissions of key directories |
| log_analyzer.sh | Reads a log file and counts errors, warnings, and failures |
| manifest_generator.sh | Takes user input and generates a structured manifest file |

## 📖 Key Git Commands Used in This Project
```bash
git init          # Initialize a new repository
git add .         # Stage all changes
git commit -m ""  # Commit with a message
git log           # View commit history
git push          # Upload to GitHub
git status        # Check current state
git branch        # List branches
```
